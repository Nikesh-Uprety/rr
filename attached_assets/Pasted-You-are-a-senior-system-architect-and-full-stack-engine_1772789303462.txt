You are a senior system architect and full-stack engineer.

Design and generate a modern, high-performance full-stack e-commerce platform called **Urban Threads**.

The goal is a production-quality application that is **fast, scalable, lightweight, and visually minimal**.

The project should be structured so that the **UI is built first**, followed by the backend APIs and database integration.

────────────────────────

TECH STACK (STRICT)

Frontend:

* Next.js 14 (App Router)
* TypeScript
* TailwindCSS
* shadcn/ui component library
* Framer Motion (animations)
* Zustand (state management)
* React Hook Form + Zod validation
* Recharts (analytics charts)

Backend:

* Next.js API routes
* Prisma ORM

Database:

* PostgreSQL (recommended for relational ecommerce data)

Authentication:

* NextAuth.js with credentials provider
* Role-based access: admin and staff

Deployment target:

* Replit / Vercel compatible architecture

Performance requirements:

* Server components where possible
* Lazy loading for heavy components
* Image optimization
* Avoid unnecessary libraries
